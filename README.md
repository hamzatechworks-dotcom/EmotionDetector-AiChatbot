# EmotionDetector-AiChatbot
📌 Project Overview :

This project combines an emotion detection model with an AI chatbot powered by TinyLLaMA.
The system first detects the user’s emotion from text and then generates an intelligent, context-aware response.

It is designed as a lightweight, efficient solution suitable for real-time conversational applications.

~ Features :
. Emotion detection from user input text
. AI chatbot with contextual responses
. Lightweight model using TinyLLaMA
. Hybrid system (ML + rule-based logic)

~ Model Details :
Emotion Detection
Dataset: GoEmotions (Hugging Face)
Model: Fine-tuned transformer (e.g., DistilBERT)
Output: Top detected emotion(s)

~ Chatbot :
Model: TinyLLaMA
Type: Lightweight LLM for fast responses
Enhancement: Rule-based responses based on detected emotion
